# Skillscore-Asish189
SkillScore is a full-stack web application designed to help institutions and organizations create, manage, and evaluate skill-based assessments efficiently. The platform provides a powerful Admin Dashboard for managing questions, categories, subtopics, and performance analytics, along with a scalable backend architecture.
SkillScore

SkillScore is a full-stack skill assessment platform that allows admins to manage MCQ-based questions and view real-time analytics through a modern dashboard.

Features

Admin dashboard with real-time statistics

Add, edit, delete MCQ questions

Store and manage correct answers

Category & subtopic-based questions

Interactive charts and analytics

Tech Stack

Backend: Java, Spring Boot, Spring MVC, JPA, Hibernate

Frontend: JSP, HTML, CSS, Bootstrap, JavaScript

Database: MySQL

Use Case

Ideal for online assessments, placement preparation, and skill evaluation systems.
